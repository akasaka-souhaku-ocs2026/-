PROFILE & STRENGTHS

プロフィール＆強み

北海道情報大学
経営情報学部 システム情報学科

システム情報学科に所属し、経営と情報の両面から課題を分析・解決するアプローチを修得。 組織の非効率なメカニズムを敏感に発見し、ITを用いたシステム化・自動化に貢献する「課題発見力」と「エンジニアリング探究力」を備えています。

STRENGTH 01

経営×情報の広範な視点

単にコードを書く枠組みを越え、「経営課題の解決にITがどうリンクするか」という広域的なレイヤーでシステム要件を吟味できる多角的発想を強みとしています。

STRENGTH 02

強固な課題発見力と変革力

「既存のアナログ慣習」を無批判に受け入れるのではなく、業務活動に存在するムダや手書きプロセスの不健全なアナログ作業を発見し、技術力によってデジタル効率化を図るマインドを持ちます。

PROJECT DETAILS

制作実績

WEB DEVELOPMENT

部活動の掲示板WEBサイト

HTML CSS

概要

連絡の遅れや周知漏れが発生しやすかった部内のアナログ連絡体制をデジタル移行するために構築した、HTML/CSSベースの掲示板サイトです。

課題の解決成果

これまで対面での口頭確認やホワイトボード、紙の回覧に依存していた部活動の連絡ステップを完全デジタル化。部員が場所を選ばず速やかに最新情報を確認できる仕組みを構築し、配布ペーパーレス化とともに、伝達ミスを劇的に削減いたしました。
